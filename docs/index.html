介绍

Dida\Db的目标是做一个轻巧的、快速的、包含常用关系数据库操作、没有复杂依赖关系的数据库管理包，用于大多数常见的数据库操作场景，它是 Dida框架 的一部分。

命名空间(Namespace)

Dida\Db所有的文件都基于 Dida\Db命名空间或者其子空间。

类名清单

类名 用途
Dida\Db\Db 逻辑数据库
Dida\Db\Connection 负责数据库的物理连接
Dida\Db\DataSet 查询结果的数据集
Dida\Db\Query 负责管理查询请求
Dida\Db\Builder 查询构造器，负责把查询请求编译成对应的SQL表达式
Dida\Db\SchemaInfo 实现Schema的查询处理
Dida\Db\Util 工具类
